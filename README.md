# Prueba Técnica - Sistema de Login en Next.js

Este es un proyecto realizado en **Next.js 13+** usando **App Router**, con un formulario de login y autenticación básica.  

## Características principales

- Login con **CI, correo y contraseña**  
- Validaciones de campos vacíos y credenciales incorrectas  
- Redirección a **menú principal** al iniciar sesión correctamente  
- Navbar con navegación entre páginas (`menu`, `about`, `login`)  
- Maquetado con **Bootstrap 5 CDN**  
- Diseño **responsive**:  
  - Escritorio → formulario centrado verticalmente y alineado a la izquierda  
  - Mobile → formulario centrado vertical y horizontalmente  
- API interna para login con **status code 200 / 401** en formato JSON  
- Mensajes de error mostrados directamente en el formulario
